Добавление артикулов из файла .xlsx


Выберите файл с расширением .xlsx.
Важно! Расположение столбцов в таблице должно быть: Артикул--Описание--КодТНВЭД--Сопроводительные Документы.
Рекомендуется скопировать только нужные ячейки в новый документ .xlsx, не копируя оглавление таблицы.
В файле не должны содержаться артикулы, которые уже имеются в БД.
